Make sure you node installed in your machine
Download the zip file and extract the files in your local system
Open the project in VS Code
Run the following commands 
npm init -y
npm i express cors multer pdf-parse fs path
After the packages were installed run the server, use node backend/server.js
In the terminal you can see ' Server running on http://localhost:3000"
Open the link http://localhost:3000 in any browser and upload the required file and hit the button to see the results
Refer the uploaded video for demo
